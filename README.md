# HDL-IDS: Hybrid Deep Learning Intrusion Detection System

## Overview
This project implements CNN + LSTM + Attention-based IDS with optional federated learning extension.

## Installation
pip install -r REQUIR~1.TXT

## Run Training
python main.py --mode train

## Run Evaluation
python main.py --mode test

## Output
- Model saved: outputs/model.h5
- Metrics: outputs/metrics.json
- Plots: outputs/

## Dataset
Download from Kaggle and place CSV in /data folder.

## Paper Alignment
Matches HDL-IDS architecture described in manuscript.
